About the Project

A Tool for Researchers

RCTrials was developed to help researchers gather survey responses from trial participants. It does not gather any personally identifying information (PII) from participants so it is completely double-blinded. It can support multiple groups, pre/during/post trial surveys, and CSV exporting for statistical analysis.

A Tool for Developers

The tool can also be used by app developers to integrate with the RCTrials API and add a randomized controlled trial (RCT) to their mobile app. The users can be divided into multiple groups, each group can be assigned specific app features for A/B testing and surveyed for feedback. Read more about the API here.

There are three different ways to use the RCTrials project — ① by using the online hosted solution at RCTrials.tk along with the free Surveyor app; ② by using the hosted solution with your own app, and integrating with the RCTrials API; or ③ by using the source code and hosting your own server with your own app.

Hosted Solution

register; create trial; create groups; create surveys & questions; inform participants of TID to use when registering and where to download the app; execute trial; observe data; export for statistical analysis;

Integrated

read the api to make sure the system can facilitate your needs; register->create...get TID; use TID in your code;

Custom Solution

fork repo (either research server or app); customize; deploy; follow steps in the hosted solution but on your own server and/or your own app.

The tool allows the researcher to define the parameters of the study - the start and end dates for the registration and trial, how many groups, to what features each group is entitled, and what surveys each group needs to complete. The system generates a four-digit code which the researcher then gives to the trial participants. When the participants submit survey answers the researcher is able to see aggregate information on the survey responses from all participants in each group.

The participants download the free Surveyor app and join the trial with the given four-digit code. The app will ask all participants to fill out the pre-test surveys, and when the trial begins split the participants into the defined groups. During and after the trial other surveys can be requested and will appear in the app for the participants in that group.
